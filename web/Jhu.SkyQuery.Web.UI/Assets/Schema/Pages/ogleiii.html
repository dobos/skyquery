The main goal of this catalog is to record all variable sources located in the OGLE-III fields in the Magellanic Clouds and Galactic bulge. The data currently available include: classical Cepheids in the Galactic Bulge, LMC and SMC,

  • type II Cepheids in the Galactic Bulge, LMC and SMC,
  • anomalous Cepheids in LMC and SMC,
  • RR Lyrae stars in the Galactic Bulge, LMC and SMC,
  • Long Period Variables in the Galactic Bulge, LMC and SMC,
  • Double Period Variables in LMC,
  • R CrB stars in LMC,
  • δ Sct stars in LMC.
